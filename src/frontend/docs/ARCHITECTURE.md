# Frontend Architecture

## Overview
This document describes the architecture of the LearningXHub frontend application, a React-based web application for practicing MCQ questions in Physics, Chemistry, and Biology.

## Technology Stack
- **Framework:** React 19 with TypeScript
- **Routing:** TanStack Router
- **State Management:** React Query (TanStack Query) for server state
- **Styling:** Tailwind CSS with shadcn/ui components
- **Authentication:** Internet Identity (ICP)
- **Backend:** Motoko canister on Internet Computer
